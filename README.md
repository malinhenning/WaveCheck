***Wave Check***
**Description**
A program that tells you where the surf is best based on your own preferences. 
Collects data of todays conditions, checking local Beaches and tells you if you can go there or if there is another place better for the conditions.

**Functions**
* Save input
* Gives you the best alternative based on conditions

**Techniques used**
* JacaScript - functions, if/else
* HTML
* CSS
